# autoAlquiler
Alquiler automático de instalaciones en la web del patronato de deportes de Dos Hermanas:
https://oelectronica.doshermanas.net/cgi-bin/inFini

Hay que modificar en el código el nombre de usuario y contraseña, además este ejemplo es para una instalación concreta en un día concreto, para variar la instalación, hora y lugar habría que inspeccionar el código de la página y establecer el XPATH deseado.
